# Трекер сроков годности

**Умное веб-приложение на чистом JS** для автоматизации контроля за сроками годности продуктов в розничной торговле. Приложение поможет сотрудникам магазинов эффективно управлять товарными запасами, минимизировать потери от просрочки и соблюдать требования законодательства.

![Скриншот интерфейса](tracker.gif)

## Почему чистый JavaScript?

Я сознательно выбрала чистый JavaScript без фреймворков и TypeScript, чтобы:

- **Понять основы на практике** - работа с DOM, событиями, асинхронностью, модулями 
- **Освоить нативный API** - IndexedDB, Date, File API
- **Создать архитектуру без фреймворков** - модули, классы, управление состоянием
- **Прочувствовать преимущества Typescript** - столкнувшись с ошибками типов в runtime
- **Научиться рефакторингу** - уменьшила main.js до 96 строк через модулизацию

## Основные функции

- Добавление продуктов (название, дата изготовления, дата окончания, категория)
- Возможность добавить фото (для распознавания товара)
- Цветовая индикация для секций и дней календаря, в зависимости от того, сколько дней осталось до оконания СГ
- Сортировка по дате и названию
- Поиск нужного продукта
- Автоматический расчет при изменении (карточка запоминает СГ товара)
- Калькулятор при открытии приложения, при добавлении и в модальных окнах, который расчитывает и переносит дату изг. и дату окончания СГ в поля ввода, что избавляет от лишних действий
- Календарь: возможность следить за товарами в определенный день, изменять их и проверять
- Секция "Архив" - дает возможность убрать и при этом сохранить товар, которого нет на полке
- Полное сохранение в вашем браузере (IndexedDB)

## Архитектура и ключевые решения

Проект использует модульную архитектуру с четким разделением ответственности.

**Ключевые решения:**

- Классовая архитектура - ModalManager, FormSearch, DateCalculator
- Ручное управление состоянием - без сторонних библиотек
- IndexedDB - хранение изображений и больших данных
- Accessibility - полная навигация с клавиатуры
